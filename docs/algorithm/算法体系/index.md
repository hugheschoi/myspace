## 五遍
- 第一遍(读题思考、看题解、背诵默写好的解法)
- 第二遍（马上自己、leetcode提交，多种解法体会，优化）
- 第三遍（过一天后，在重复做题，不同解法的熟练程度，专项练习）
- 第四遍（过一周后，反复练习相同题目）
- 第五遍（面试前一周恢复性训练）

## 切题
1. 审题与出题者沟通，是否理解正确
2. 想出所有解题的办法，比较写出最优解
3. 多写，多敲代码
4. 测试用例也写上

## 复习和观看指南
1. 题目不熟的时候，一定要先看好题目，花几分钟思考，5分钟还想不出，停止死磕，看题解
2. 复习时专注一分钟以上观看题解！
